# 网络流

       很多问题都可以转行为二分图匹配，所以二分图匹配的匈牙利算法要掌握。在网络流中有四个概念需要强调：
    
    匹配：在G中两两没有公共端点的边集合M
    边覆盖：G中的任意顶点都至少是F中某条边的端点的边集合F
    独立集：在G中两两互不相连的顶点集合S
    顶点覆盖：G中的任意边都有至少一个端点属于S的顶点集合S
    
       以上四个概念满足如下关系：
    
    对于不存在孤立点的图，|最大匹配|+|最小边覆盖|=|V|
    |最大独立集|+|最小顶点覆盖|=|V|
